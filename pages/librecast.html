LibreCast

Free on github
LibreCast is a work-in-progress multi platform client developed as a school project. It allows users to stream videos, manage playlists and subscriptions, browse online content and more. The desktop is maintained by 3 students, and a web version is currently being made by another student. Both are currently only available in French.
The source code is publicly available at github.com/LibreCast/LibreCast.
